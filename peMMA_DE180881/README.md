
# Product Management & Shopping Cart App (Expo + SQLite)

## How to run
1. Install dependencies:
   - `npm install` or `yarn`
   - This project assumes Expo. You need `expo-cli` installed globally.
2. Start Expo:
   - `expo start`
3. Note: This scaffold uses `expo-sqlite` and `@react-native-async-storage/async-storage`.

## Project structure
- App.js - entry
- /screens - screens for app
- /components - reusable components
- /db - sqlite helper
- /assets - images
